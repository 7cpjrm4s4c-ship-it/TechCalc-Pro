## Consolidated Changes

### Phase  34B COMPONENTS CSS REBUILD

`css/components.css` wurde von einem historisch gewachsenen Patch-Stack zu einer kompakten, globalen Komponentenbasis zurückgebaut. `components.css` enthält nur noch wiederverwendbare UI-Komponenten. Modulklassen wie `hx-*`, `wrg-*`, `pipe-*`, `wastewater-*`, `rainwater-*` und `buffer-*` werden in `modules.css` geführt, solange sie nicht vollständig in `tc-*` Primitives migriert sind.

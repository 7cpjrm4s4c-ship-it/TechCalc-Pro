# Phase 36A.12 – Saved Records 1:1 wie Heizung/Lüftung

## Änderung
Regenwasser und Schmutzwasser verwenden für die Saved-Record-Rows jetzt denselben Dynamic-Island-Vertrag wie Heizung/Lüftung:

```html
data-line-dynamic="line-sections"
```

statt:

```html
data-platform-dynamic="saved-records"
```

## Erwartung
- Speichern bleibt stabil
- Markieren bleibt stabil
- Aktualisieren bleibt stabil
- Löschen bleibt stabil
- Accordion bleibt nach einfachem Tipp geöffnet
- Scroll-Hänger beim offenen Accordion sollten reduziert sein

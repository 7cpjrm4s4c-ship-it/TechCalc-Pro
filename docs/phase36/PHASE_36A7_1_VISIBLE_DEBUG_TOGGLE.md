# Phase 36A.7.1 – Visible Debug Toggle

## Änderung
Die Debug Card ist in Schmutzwasser und Regenwasser immer sichtbar.

## Bedienung
- `Debug anzeigen` aktiviert das Mitschreiben.
- `Debug leeren` leert die Events.
- `Debug ausblenden` deaktiviert das Mitschreiben.

Kein URL-Parameter und keine Browser-Konsole mehr erforderlich.
